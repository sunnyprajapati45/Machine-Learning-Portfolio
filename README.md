# 🧠 Machine Learning Portfolio

## 📌 Problem Statement
Applying theoretical Machine Learning knowledge to real-world datasets is critical. This repository acts as a centralized portfolio demonstrating practical implementations of various predictive and analytical models.

## 🚀 Solution Approach
Contains a suite of models including Heart Disease Prediction, house pricing models, and classification tasks. The notebooks follow a structured approach: Exploratory Data Analysis (EDA) -> Data Preprocessing -> Feature Engineering -> Modeling -> Evaluation.

## 🛠️ Tech Stack
- **Language:** Python
- **Libraries:** Scikit-Learn, Pandas, NumPy, Matplotlib, Seaborn
- **Environment:** Jupyter Notebooks

## ✨ Features
- End-to-end regression and classification models
- Comprehensive Exploratory Data Analysis (EDA)
- Implementation of multiple algorithms (Random Forest, SVM, Logistic Regression)
- Deep focus on evaluation metrics (Precision, Recall, F1-Score)

## ⚙️ How to Run
```bash
git clone https://github.com/sunnyprajapati45/Machine-Learning-Portfolio.git
cd Machine-Learning-Portfolio
jupyter notebook
```
